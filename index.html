¿Que nos Motiva?

Nuestra motivación es que el problema es que la gente pueda saber que cantidad de material requiere para realizar un proyecto de remo delación o de construcción nuevo, y poder apoyar en su economía y pueda llevarlos a cabo,Nosotros deseamos poder emplear los conocimientos que estamos adquiriendo para elaborar un programa que sea sencillo y con un problema particular y a partir de este podamos enfocarnos en problemas mas complejos.

¿Por qué Nuestro Software?

La implementación de este software resuelve el problema, partiendo que se obtiene el área y se describe la cantidad necesaria de material que requiere en especifico, con un rango de desperdicio y de esta forma las personas no gasten mas allá de lo necesario para elaborar sus proyectos.

Descripción del programa

El programa presenta un menú de opciones dónde el usuario selecciona el tipo de proyecto que quiere ejecutar, después se piden los datos para calcular el área y una vez determinada obtiene la lista de materiales así como los precios aproximados de cada material
